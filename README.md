# naswek-homepage

Маленькая домашняя страница, которая сейчас отдается простым Go HTTP-сервером.

Пока это минимальный проект: статический `index.html`, несколько ссылок и health-check.

## Технологии

- Go 1.26
- стандартный `net/http`
- статическая HTML-страница в `static`

## Запуск

```bash
go run .
```

После запуска:

- сайт: `http://127.0.0.1:8080`
- health-check: `GET /health`

## Структура

- `main.go` - запуск HTTP-сервера;
- `static/index.html` - главная страница;
- `go.mod` - Go-модуль проекта.

## На будущее

Планируется развивать проект не просто как страницу, а как небольшой личный hub:

- сделать нормальный дизайн главной страницы;
- добавить сервис авторизации и аутентификации;
- подключить личный аккаунт и защищенные страницы;
- добавить свои сервисы и ссылки на них;
- сделать статистику по этим сервисам;
- показывать статус сервисов: работает, недоступен, в разработке;
- добавить админку для управления ссылками, сервисами и описаниями;
- собрать все части проекта в единую систему через Docker Compose;
- позже добавить API для интеграции между сервисами.
